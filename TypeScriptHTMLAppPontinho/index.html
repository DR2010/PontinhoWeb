Algumas regras do jogo

Vamos te explicar como este jogo funciona. Estas sao as regras e modo de jogar:

1) O jogo tem que ser criado. Pontinho e' o unico jogo disponivel neste momento (em fase de construcao).

2) Depois de "Criar" um jogo, as cartas serao adicionadas ao jogo, neste caso serao 2 baralhos.

3) Selecionar jogadores para o jogo.

4) Definir o coringa.

5) O sistema definira quem comeca o jogo. (Inicialmente random, depois sera uma sequencia)

Algumas Regras:

1) "Preciso!" - O jogador tera' que pressionar o botao "Preciso" caso queira a carta que acabou de ser descartada.

O sistema vai esperar alguns segundos para outro jogador, que esta mais proximo do jogador da vez, se manifestar.

Se nenhum se manifestar a carta sera passada para o jogador e o sistema vai espera o jogador baixar a carta.

Se a carta nao for baixada o jogador sera'punido:... nao sei como ainda...

2) Coringa/ Joker - Cada jogo tera um coringa.
